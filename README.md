# konami-cb

A lightweight script that will execute a callback function if a user types in the Konami Cheat code into the browser: 

`up, up, down, down, left, right, left, right, b, a`

**Install:**

`npm i konami-cb`

**Usage:**

```js
var konami = require('konami-cb');

konami(function() {

    //- make stuff happen

});
```
